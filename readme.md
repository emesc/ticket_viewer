# Zendesk Ticket Viewer

## Created and tested using

- Ruby 2.3.1
- OS Sierra 10.12.5

## Install the needed gems using bundle

```
  $ bundle install
```

## Run the tests

For individual test file:

```
  $ bundle exec rspec spec/client_spec.rb
```

To run all the tests:

```
  $ bundle exec rspec
```

## How to use

```
  $ ./bin/zendesk
  $ menu
  $ next
  $ prev
  $ page 1
  $ show 1
  $ load
  $ quit
```
