# Zendesk Ticket Viewer

## Tools used

- Ruby 2.3.1
- OS Sierra 10.12.5

## Run the tests

For individual test file:

```
  $ rspec spec/client_spec.rb
```

To run all the tests:

```
  $ rspec
```

## How to use

```
  $ ./bin/zendesk
  $ menu
  $ next
  $ prev
  $ page 1
  $ show 1
  $ load
  $ quit
```
